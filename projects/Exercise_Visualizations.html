Exercise Visualizations

An individual project to practice some basic data visualization techniques.

Role: Data Visualization Analyst
Tools: R, RStudio

The Visualizations

I've been using the same app to track my strength training workouts for almost two years, but I realized it lets me export my data to a .csv file! I thought I'd try using R to visualize some trends in my workouts, and to help keep me motivated and pushing for progress each month.

Check out the code here on Github.

I just started a new benching routine, so I thought that data would be a good place to start practicing with the ggplot() function. It's a three day per week routine, and I do fewer reps with higher weight as the week goes on. First it's sets at 70% of my "one-rep max" on Monday, then 75% on Wednesday, and 80% on Fridays.

Right now it's hard to tell if there is an upward trend with all the plotted lines. Next up I'll examine plotting them all separately and including some markers for where I bumped my "one-rep max" to help make sense of all the dips in this plot.
